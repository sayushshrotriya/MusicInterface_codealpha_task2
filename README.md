
# 🎶 Simple Music Player

### A minimalist web-based audio player built with pure HTML, CSS, and JavaScript.

Welcome to the **Simple Music Player**, a clean and straightforward web application designed to demonstrate fundamental front-end development skills. This project provides a fully functional music interface with a sleek design and core playback controls, perfect for anyone looking to understand the basics of building a web app from scratch.

-----


### 🔗 Live Demo

You can view a live version of this project here: [Live Demo Link](https://sayushshrotriya.github.io/MusicInterface_codealpha_task2/)




### ✨ Features

  * **Responsive & Clean UI:** A modern, minimalist design that looks great on any screen size.
  * **Core Playback Controls:** Easily play, pause, and replay tracks with a single click.
  * **Intuitive Interface:** The simple layout makes it easy for any user to navigate and control the music.
  * **Lightweight & Fast:** Built with vanilla JavaScript, this player is fast and doesn't rely on any heavy frameworks.
  * **Initial Tracklist:** Comes pre-loaded with a small selection of MP3 files to get you started immediately.

-----

### 🚀 Getting Started

Getting this music player up and running is as easy as a few simple steps. You don't need to install any dependencies or frameworks\!

1.  **Clone the repository:** Grab a copy of the project files from GitHub.
    ```bash
    git clone https://github.com/sayushshrotriya/MusicInterface_codealpha_task2.git
    ```
2.  **Navigate to the project folder:**
    ```bash
    cd MusicInterface_codealpha_task2
    ```
3.  **Open in your browser:** Simply open the `index.html` file directly in your favorite web browser (like Chrome, Firefox, or Edge).

That's it\! The player should load instantly, and you'll be able to start playing music right away.

-----

### 📂 Project Structure

This project is organized to be simple and easy to understand.

  * `index.html`: The heart of the player. It defines the structure and layout of the interface using standard HTML5.
  * `style.css`: All the visual magic happens here. This file contains the custom CSS to make the player look clean and elegant.
  * `script.js`: The brain of the operation. This JavaScript file handles all the logic for playing audio, updating the UI, and managing playback states.
  * `*.mp3`: The audio files that the player loads and streams. You can easily add your own\!
